Snow mask inference

Upload an image and run the segmentation CNN

This demo uses the trained UNet-like snow segmentation model from the training directory. For best results, upload a Sentinel-style 6-band patch file such as `.npz` with an `image` array. Standard image files will still run through a fallback preprocessing path for quick testing.

Select an image
The backend returns the predicted snow mask and an overlay. If you upload a regular RGB image, the model still runs, but the result is only a rough proxy because the network was trained on 6-band satellite data.
